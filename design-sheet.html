— Essay · 04.2026

The shape of a forgotten thing.

Memory is not storage. It is a posture the mind holds toward its own past — and so is forgetting.

Aryaman Agrawal · 14 min read

Plato's wax, Camus's absurd, the cortical column — three attempts to describe the same problem from incompatible vocabularies. What if the problem is not storage at all, but the way an agent decides what is worth keeping? The answer, I'll argue, is not algorithmic. It is closer to taste.

Single-accent system: cyan #6fb3c4 only, used for links, CTAs, hover states, and meta. Analogous to the bg (low tension, quietly cold) — coherent with the observatory/quantum theme; no warm/cool clash. Contrast: text 15.9:1 (AAA), cyan 8.1:1 (AAA), muted 5.6:1 (AA with headroom). Bg #0c0d10 is neutral charcoal — neither blue nor brown, so photographs don't take on a color cast. Warm-on-cool editorial pairing (text #ece6d6 on near-black) follows NYT dark, iA Writer, Are.na, Craig Mod. Exception: warm orange (#d97706) appears only inside analog-medium frames (filmstrip frame metadata) as part of the medium itself, never as a system accent. Warm cream paper (#ece6d6) inside the lit-notebook page is similarly contained.

Photo essay — “San Francisco, in fog” curated column · serif captions · Flip lightbox on click
Photo Essay · 02.2026 · 14 frames

San Francisco, in fog.

A small set, made on early walks between November and February. Most mornings the fog reaches Divisadero by six, and is gone by nine.

There is a quietness to a city whose edges are missing. You feel the geometry without seeing it — the bridge is more present in fog than out of it. Camus would have written about this.

Motion plan: clip-path reveal on each figure as it scrolls in (inset(100% 0 0 0) → inset(0), 600ms ease). Click any figure → GSAP Flip captures grid state, repositions to fullscreen lightbox, the rest of the page dims. Esc / click-outside reverses. Reduced-motion skips the reveals, keeps the lightbox.

05 — Search & agent discoverability

Built to be cited, not just ranked

In 2026 the audience is split: traditional search (Google, Bing) and LLM agents (ChatGPT search, Perplexity, Claude, Google AI Overviews). Bing Webmaster is non-negotiable since it powers ChatGPT. Below is the day-one foundation we'll bake into the Astro scaffold — most are zero-config integrations.

Day-one checklist ship before first post

Foundations · classic SEO

  1. @astrojs/sitemap → auto /sitemap-index.xml
  2. robots.txt referencing sitemap; explicitly allow GPTBot, ClaudeBot, PerplexityBot, Google-Extended (opt-in to citation)
  3. @astrojs/rss → /rss.xml and /feed.json (JSON Feed 1.1 — agents prefer it over XML)
  4. Canonical <link rel="canonical"> on every page; Open Graph + Twitter summary_large_image defaults in base layout
  5. Semantic HTML: one <h1>, <article> per post, <time datetime> for dates

Agent discoverability · LLMO / GEO

  1. JSON-LD per page — BlogPosting on essays, Person on /about, WebSite with SearchAction on root. This is what LLMs and Google AI Overviews actually parse.
  2. Serve raw markdown at predictable URLs (/posts/[slug].md) — Anthropic and Astro docs do this; highest-ROI move for agent fetchability.
  3. /llms.txt generated from the content collection (cheap; ~10% adoption, no major crawler confirms reading it, but trivial — ship and forget)
  4. Bing Webmaster Tools + Google Search Console — submit sitemap day one. Bing is the gateway to ChatGPT search citations.
  5. IndexNow ping on publish (Bing/Yandex; trivial Astro integration)

Content patterns that get cited

  • TL;DR / summary at the top of every essay
  • Declarative claims with dates and named sources
  • Glossary-style definitions for terms (cortical memory, etc.)
  • Consistent author entity across pages — same Person schema, same bio, same canonical name
  • Recency: LLMs favor freshest version. Frequency > ranking — only ~30% of brands persist between consecutive AI answers.
One-stop tool: @jdevalk/astro-seo-graph auto-generates JSON-LD graph + llms.txt + IndexNow pings in a single integration. Worth evaluating before hand-rolling.
06 — Motion

Restraint where it counts (06)

Per your call: no animation on text content. Reserve GSAP for: home page entrance, photography interactions, section transitions, and one signature moment per landing page. Tokens: ease cubic-bezier(.2,.6,.2,1), durations 220ms (UI) / 600ms (reveal) / 1200ms (signature). Reduced-motion always honored.
